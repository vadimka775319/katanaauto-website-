# Katana Auto — Сайт

Современная версия сайта katanaauto38.ru с обновлённым дизайном.

## Запуск локально

1. Откройте файл `index.html` в браузере (двойной клик или перетащите в окно браузера)
2. Или запустите локальный сервер:
   - **Python:** `python -m http.server 8000`
   - **Node.js:** `npx serve .`
   - Откройте в браузере: http://localhost:8000

## Структура проекта

- `index.html` — главная страница
- `styles.css` — стили
- `script.js` — интерактивность (меню, каталог, фильтры)

## Деплой на ihor

После готовности загрузите все файлы на сервер в корень сайта.
